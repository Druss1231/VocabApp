# Vocabulary app for TOEIC
## Overview
A vocaburaly app designed to help people improve thier TOEIC score by providing multiple sentences using Google GEMINI API.

## Features
- Custom word list creation
- Quiz and review modes
- Genarate example sentences using Google GEMINI API

## Target Users
- General TOEIC leaner

## Tech Stack
- Frontend: React Native
- Backend: Firebase
- Hosting: AWS Amplify
- LLM Integration: Google Gemini API
